perl-Snort-Rule - Perl extension for dynamically building snort rules

Website: http://search.cpan.org/dist/Snort-Rule/
License: CHECK(GPL+ or Artistic)
Vendor: CC/Lab
Description:
This is a very simple snort rule object. It was developed to allow for
scripted dynamic rule creation. Ideally you could dynamically take a list
of bad hosts and build an array of snort rule objects from that list. Then
write that list using the string() method to a snort rules file.
