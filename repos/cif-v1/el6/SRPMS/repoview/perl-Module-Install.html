perl-Module-Install - Standalone, extensible Perl module installer

Website: http://search.cpan.org/dist/Module-Install/
License: GPL+ or Artistic
Vendor: CC/Lab
Description:
Module::Install is a package for writing installers for CPAN (or CPAN-like)
distributions that are clean, simple, minimalist, act in a strictly correct
manner with ExtUtils::MakeMaker, and will run on any Perl installation
version 5.005 or newer.
